# comch

Channel Based Communication over Network for Golang

Usage :

```go
package main

import (
	"fmt"

	comch "github.com/husamettinarabaci/comch"
)
```

## Installation:

```shell
go get "github.com/husamettinarabaci/comch@latest"
```
